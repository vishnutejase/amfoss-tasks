## TERMINAL COMMANDS

1. **cd:**
 Used to change directories.

2. **touch:**
 Used to create empty files.

3. **mkdir:**
 Used to create a new directory.

4. **ls:**
 Used to list contents of a directory.

5. **echo \<base64encodedString\> | base64 --decode:**
 Used to decode and encoded.

6. **cat:**
 To print out content of specific files.

6. **vim:**
 To open a file in vim.

## GIT COMMANDS

1. **git clone:**
 Used to clone a repository.

2. **git checkout \<branch name\>:**
 To switch between branches in the repository.

3. **git init:**
 Creates a new git repository.

4. **git add -A:**
 Adds all files for staging.

5. **git status:**
 Shows the state of the working directory.

6. **git commit -m "{commit name}":**
 Commits changes and additions, creating a log.

7. **git push -u origin master:**
 Pushes changes to the repository.
